Better User Experience for Browser Zoom Users

In general, the thing to avoid the most is tedium caused by too many nested scroll bars

  • For predominantly vertical layouts, there should be only one vertical scroll bar
  • For predominantly horizontal layouts (rare), there should be only one horizontal scroll bar

For most text, the expectation is that text gets bigger and flows.

For most content images, the expectation is that they get bigger, presenting a horizontal scroll bar if two dimensions are necessary for the images value to the user. e.g. maps, some graphics, etc.
